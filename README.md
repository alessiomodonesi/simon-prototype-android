# simon-prototype-android
Progetto di Sistemi Embedded 2025/2026

## Dispositivo di Sviluppo e Test

Come richiesto dalle specifiche del progetto, l'applicazione è stata sviluppata e testata utilizzando il seguente dispositivo virtuale:

* **Tipo di dispositivo:** Emulatore (Android Virtual Device)
* **Modello:** Google Pixel 2
* **Versione Android:** API Level 36.0
* **Risoluzione Schermo:** 1080 x 1920 px
* **Densità Schermo:** 420 dpi
* **RAM:** 4 GB (4096 MB)
